# Wave Survival Boss Fight

Аркадный 2D survival-проект на Pygame: отбивайтесь от волн врагов, улучшайте оружие и побеждайте боссов.

## Краткое описание
Wave Survival Boss Fight — это игра в жанре «волны противников» с боссами, улучшениями оружия, магазином и безопасной хижиной между волнами. Игра написана на Python с использованием Pygame и поддерживает локализацию (английский, русский, украинский).

## Возможности
- Несколько карт/тем с уникальной атмосферой (Forest, Winter, Inferno, Swamp, Night City).
- Волновая система: волны с обычными врагами, мини-боссами и боссами.
- Магазин оружия и безопасный дом (house) между волнами.
- Система апгрейдов оружия (урон, обойма, скорость перезарядки).
- Подбор патронов и аптечек (drop system).
- Музыка и SFX с регулировкой громкости.
- Поддержка полноэкранного режима и смены языка.
- Возможность собрать с помощью PyInstaller (в репозитории есть `main.spec`).

## Структура проекта
Основные файлы:
- `main.py` — точка входа, инициализация Pygame и запуск цикла.
- `constants.py` — константы, конфигурация карт, врагов и локализация.
- `assets.py` — загрузка ресурсов (изображения, TMX карт), генерация оружия/текстур.
- `game_loop.py` — игровая логика: движение, стрельба, спавн врагов, бои с боссами.
- `menus.py` — меню, магазин, настройки и интерфейсы.
- `utils.py` — вспомогательные функции (шрифты, локализация, отрисовка).
- `save_data.py` — загрузка/сохранение прогресса (`save.json`).
- `assets/` — директория с графикой/аудио/картами.

Подробное описание структуры есть в `PROJECT_STRUCTURE.md`.

## Требования
- Python 3.8+ (или 3.x)
- Pygame (рекомендуется последняя стабильная версия)

Установка зависимостей:
```
pip install pygame
```

## Запуск
В корне репозитория:
```
python3 main.py
```
Если хотите запускать в оконном режиме, зайдите в Настройки в главном меню и отключите Fullscreen.

## Управление (по умолчанию)
- Движение: W A S D
- Стрельба: стрелки (Up / Down / Left / Right) — удерживайте клавишу направления для выстрелов
- Перезарядка: R
- Использовать аптечку: 1
- Вход в дом/интерактив: E (при подходе к дому во время паузы между волнами)
- Открыть чат для ввода чит-кодов: T
- Пауза/меню: ESC
- Подтверждение в меню: Enter
- Навигация в меню: стрелки или мышь

Есть чит-коды для пропуска уровней: `SKIPLEVEL`, `NEXTLEVEL` (вводятся в режиме чата или по последовательности клавиш).

## Сохранения и настройки
- Прогресс и настройки сохраняются в `save.json`.
- В настройках можно менять громкость музыки и звуков, включать/выключать полноэкранный режим и выбирать язык (en/ru/uk).

## Работа с ассетами
- Основные ассеты (изображения, звуки, карты) находятся в папке `assets/`.
- Карты собираются через TMX (Tiled). Функция загрузки TMX есть в `assets.py`.
- Игровые части игрока загружаются из `assets/player_parts/` (если отсутствуют — игра использует запасные спрайты).

## Как вносить правки и развивать проект
- Добавление оружия: обновите `assets.WEAPONS` и положите соответствующие спрайты в `assets/weapons/`.
- Настройка врагов/волн: изменяйте `constants.ENEMY_TYPES`, `MAP_THEMES` и волновые конфигурации в `constants.py`.
- Добавление локализации: допишите ключи в `constants.TEXT` для новых языков и добавьте код в `constants.LANGUAGES`.

## Упаковка
В репозитории есть `main.spec` — можно использовать PyInstaller для создания standalone-­сборки:
```
pyinstaller --onefile main.py
```
(проверьте и доработайте spec-файл для корректного включения папки `assets`)

## Советы по отладке
- Если игра падает с ошибкой загрузки ресурсов — проверьте, что папка `assets` содержит нужные файлы (карты, изображения, sound/map.mp3).
- Для отладки можно запускать с выводом в консоль (ошибки загрузки шрифтов/аудио уже логируются).

## Известные детали реализации
- Локализация управляется через `utils.tr(key)` и `constants.TEXT`.
- Визуальные темы карт применяются в `assets.create_themed_map_surface`.
- Система апгрейдов использует `constants.UPGRADE_COSTS` и хранит уровни в `constants.weapon_upgrades`.
- Сохранения хранятся в `save.json`; формат можно просмотреть/редактировать вручную (JSON).
